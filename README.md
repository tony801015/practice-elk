# practice-elk(docker-compose)

The practice create simple and speed to usage ELK.


## Usage

### start
```
bash start.sh
```

127.0.0.1:5601 kibana 
127.0.0.1:9200 elasticsearch
127.0.0.1:8080 nginx

### stop
```
bash stop.sh 
```
